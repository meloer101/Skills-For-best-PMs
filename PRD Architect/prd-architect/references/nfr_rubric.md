# Non-Functional Requirements Rubric

Keep NFRs measurable and scoped.

## Performance

- Latency: p95 response time target
- Throughput: requests per second or jobs per hour
- Payload limits: max size per request

## Reliability and Availability

- Uptime target (monthly or quarterly)
- Error budget and retries
- Graceful degradation behavior

## Security and Privacy

- Authentication and authorization requirements
- Data classification and storage rules
- Audit logging needs

## Scalability and Limits

- Expected peak users or traffic
- Rate limits and quotas
- Backpressure strategy
