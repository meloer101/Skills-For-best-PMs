# Experiment Catalog

Pick the smallest test that reduces the biggest uncertainty.

## Types

- A/B test: compare variants on a primary metric
- UX simplification test: remove steps to reduce drop-off
- Messaging test: change copy to improve activation
- Targeted rollout: enable for a segment to validate value
- Concierge test: manual workflow to validate behavior

## Required Fields

- Hypothesis
- Target segment
- Success metric and threshold
- Duration and sample size (if known)
